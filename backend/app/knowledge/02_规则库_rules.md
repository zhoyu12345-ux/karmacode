## 第一部分：婚姻情感规则 (Marriage & Relationships)

### 1.1 女命婚姻核心规则 (Core Rules for Female Charts)

**总纲（《三命通会·论女命》）：**
> "妇人何利？利在夫星。夫利，其妇必利；夫困，其妇必困。"

**规则1.1.1 — 夫星纯度原则**
- 八字中正官或七杀，有且仅有一种为吉（"纯"）
- 官杀混杂 → 感情复杂，易有多段感情纠葛
- 判定：天干透出正官+七杀 = 官杀混杂
- 例外：若一者被合化或制化，则不算混杂

**规则1.1.2 — 夫星旺衰判断**
- 夫星得月令（生于官杀旺的月份）→ 丈夫有能力
- 夫星坐旺地（如正官坐禄）→ 婚姻质量高
- 夫星临死绝之地 → 丈夫平庸或身体欠佳
- 夫星被伤官克制 → 婚姻波折，性格冲突

**规则1.1.3 — 八法断命体系**
| 格局 | 判断条件 | 婚姻解读 |
|------|----------|----------|
| 纯 (Pure) | 独一位官或杀，无混杂，有财有印 | 贤妻良母，夫荣子贵 |
| 和 (Harmonious) | 日主柔弱，独一夫星，中和无破 | 恬静温和，益夫旺子 |
| 清 (Clean) | 官杀不混，夫星得时，财生官 | 清贵之命，旺夫兴家 |
| 贵 (Noble) | 官星得财气相资，带三奇贵人 | 极品贵命 |
| 浊 (Murky) | 五行失位，正夫不显，偏夫丛杂 | 感情混乱，婚姻艰辛 |
| 滥 (Overflow) | 明暗夫星过多，带凶煞 | 克夫再嫁之象 |
| 娼 (Harlot) | 身旺夫绝，食神过旺制官 | 婚姻不顺，独身倾向 |
| 淫 (Lustful) | 日主自旺，明暗夫星交集过多 | 感情泛滥，难以专一 |

**规则1.1.4 — 旺夫克子 / 旺子伤夫 判断**
- 夫星生旺 + 子星衰败 → 旺夫克子
- 子星得地 + 夫星失令被克 → 旺子伤夫
- 两者皆失 → 夫子皆不得力
- 两者皆旺 → 福寿双全

### 1.2 男命婚姻核心规则 (Core Rules for Male Charts)

**规则1.2.1 — 妻星判断**
- 正财 = 正妻
- 偏财 = 偏房/情人
- 正财透干而被合 → 妻子可能被他人"合走"（出轨风险）
- 比劫重重克财 → 克妻或晚婚
- 财星入墓 → 妻子身体欠佳或婚姻压抑

**规则1.2.2 — 妻宫判断**
- 日支（妻宫）：
  - 坐正财 → 娶贤惠之妻
  - 坐偏财 → 妻子精明能干
  - 坐比劫 → 易有第三者
  - 坐印星 → 妻子如母亲般照顾
  - 坐伤官 → 夫妻易争吵
  - 被冲（子午冲等）→ 婚姻不稳定
  - 被合 → 关系融洽

### 1.3 合婚规则 (Compatibility Matching)

**规则1.3.1 — 五行互补原则（最高优先级）**
```
男方过旺五行 → 宜配女方克制该五行 → 绝佳婚配
例：男方木太旺 → 宜配金多（金克木）或火多（火泄木）之女
```

**规则1.3.2 — 日柱合婚法（核心）**
- 日干相合（甲己合、乙庚合、丙辛合、丁壬合、戊癸合）→ 夫妻同心
- 日支（夫妻宫）互不刑冲克害 → 基本匹配
- 日柱天克地冲 → 婚姻大忌，极难白首

**规则1.3.3 — 合婚判定优先级**
1. 五行互补 > 2. 十神互补 > 3. 日柱不冲 > 4. 四柱合多 > 5. 用神互益 > 6. 纳音参考 > 7. 属相参考

**规则1.3.4 — 硬配硬原则**
- 所谓"克夫"命配"克妻"命 → 反而彼此制衡
- "比劫重"配"食伤多" → 互相调和化解

### 1.4 桃花/咸池规则 (Peach Blossom / Romance Indicators)

**规则1.4.1 — 桃花位查找**
```
寅午戌年/日生 → 桃花在卯
巳酉丑年/日生 → 桃花在午
申子辰年/日生 → 桃花在酉
亥卯未年/日生 → 桃花在子
```

**规则1.4.2 — 桃花解读**
- 桃花在生旺之地 → 容貌俊美，重感情，浪漫多情
- 桃花在死绝之地 → 感情不检点，游荡

- 桃花 + 羊刃 → 因色惹祸
- 桃花在日时柱 → 影响最重
- 女命带桃花 → 需看整体格局，不可简单论凶

**规则1.4.3 — 桃花时机**
- 大运/流年走到桃花位 → 桃花运开启
- 桃花逢合 → 恋情确定
- 桃花逢冲 → 感情波动/分手

### 1.5 婚姻时机判断 (Marriage Timing)

**规则1.5.1 — 结婚大运判断**
- 大运走到夫星/妻星旺地 → 结婚概率高
- 大运走到夫妻宫（日支）被引动 → 婚姻之期
- 大运天地合日柱 → 婚期信号

**规则1.5.2 — 结婚流年判断**
- 流年天干为夫星/妻星 → 遇到正缘之年
- 流年地支合动日支（夫妻宫）→ 大概率结婚
- 流年桃花星出现且为喜用 → 恋爱结婚之象
- 流年天喜/红鸾星出现 → 喜庆之年

## 第二部分：财富规则 (Wealth Rules)

### 2.1 正财规则 (Earned Income)

**规则2.1.1 — 正财格基本判断**
```
《三命通会》："正财者，乃甲见己、乙见戊之类。受我克制，为我之妻。"
```

- 正财得时乘旺 + 日主有力 → 稳定富足
- 财多身弱 + 无印助 → 富屋贫人（看得到用不了）
- 财少身强 + 有比劫 → 收支勉强平衡

**规则2.1.2 — 正财藏露**
- 正财藏于地支 → 家资丰厚，不显山露水（藏则丰厚）
- 正财透于天干 → 财气外露，容易消耗（露则浮荡）
- 正财透出+有比劫 → 宜透出使人共见，劫财不敢夺

**规则2.1.3 — 财印顺序**
> 《独步》："先财后印，反成其福；先印后财，反成其辱。"
- 大运先走财运后走印运 → 先赚钱后置产，富贵之道
- 大运先走印运后走财运 → 先安逸后求财，反受其辱

### 2.2 偏财规则 (Unexpected Fortune)

**规则2.2.1 — 偏财格判断**
```
《三命通会》："偏财者，乃甲见戊，乙见己之例。非妻所带，乃众人之财也。"
```

- 偏财透出 → 慷慨大方，轻财好义
- 偏财 + 官星 → 以财生官，富贵双全
- 偏财被比劫分夺 → 钱来钱去，存不住
- 偏财落空亡 → 求财不得，官运受阻

**规则2.2.2 — 偏财来源**
- 月上偏财 → 继承家业或外祖产业
- 年上偏财 → 祖上荫庇
- 日下偏财 → 自己打拼的意外之财
- 时上偏财 → 晚年暴富或子女带来财富

### 2.3 综合财富判断

**规则2.3.1 — 财旺生官**
- 财星旺盛 + 官星出现 → 以财富换取地位，富而且贵
- 露官藏财 → 高位厚禄

**规则2.3.2 — 财星忌讳**
- 财星被合 → 财富被人分走
- 财星空亡 → 求财无果
- 财星逢冲 → 财运波动剧烈
- 比劫夺财 → 赚到钱就被分走

## 第三部分：每日决策规则 (Daily Decision Rules)

### 3.1 流日十神判断法

**核心规则：以流日天干对应日主十神，判断当日能量主题**

| 流日十神 | 能量主题 | 宜 (Auspicious) | 忌 (Caution) |
|-----------|----------|-----------------|--------------|
| 正财/偏财 | 财富流动 | 处理财务、签约、收款、购物 | 考试、读书、长辈事务 |
| 正官/七杀 | 事业压力 | 承担责任、汇报、晋升、官方事务 | 高风险投资、冒险 |
| 正印/偏印 | 学习静修 | 读书、规划、体检、寻求建议 | 身旺者浑浑噩噩 |
| 比肩/劫财 | 社交竞争 | 聚会、社交、交换意见 | **求财、投资、借钱出去** |
| 食神/伤官 | 创意表达 | 创作、演讲、学习新技能、美食 | 得罪人、挑战权威 |

**决策口诀：**
```
比劫之日莫求财，访友聚会可畅怀。
食伤有口福，官杀压力来。
财星日里钱财动，印星静修智慧开。
```

### 3.2 流日特殊信号

**规则3.2.1 — 天克地冲日（高危）**
- 流日天干克日柱天干 + 流日地支冲日柱地支 = 天克地冲
- 表现：突发急事、车祸风险、重大争执
- 建议：避免高危活动，不做重大决策

**规则3.2.2 — 伏吟日（郁闷）**
- 流日干支与日柱完全相同
- 表现：诸事不顺，心情郁闷
- 建议：低调行事，不宜做重要决定

**规则3.2.3 — 三合/三会日（能量极强）**
- 流日+流月+大运形成三合局 → 能量极强
  - 为喜用神 → 巨大好运
  - 为忌神 → 重大不顺

### 3.3 择日原则

1. 选择用神日做重要决策
2. 避开忌神日做高风险活动
3. 避开天克地冲日
4. 财星日适宜商业活动
5. 印星日适宜学习规划

## 第四部分：八字推算算法规则 (Calculation Rules)

### 4.1 年柱确定
- 以**立春**为年柱分界点（非农历正月初一）
- 立春前 → 上一年干支
- 立春后 → 新一年干支

### 4.2 月柱确定
- 以**节气**为月柱分界点
- 寅月 → 立春到惊蛰
- 卯月 → 惊蛰到清明
- 依此类推...
- 月干用**五虎遁**推算：甲己之年丙作首，乙庚之岁戊为头...

### 4.3 日柱确定
- 公历日期 → 干支纪日（万年历查询或公式计算）

### 4.4 时柱确定
- 时辰对应地支：23-1子时、1-3丑时...
- 时干用**五鼠遁**推算：甲己还加甲，乙庚丙作初...

### 4.5 真太阳时校正
- 北京时间 + (出生地经度 - 120°) × 4分钟

### 4.6 十神推算
```
以日干为"我"：
同我者为比劫（同性比肩，异性劫财）
我生者为食伤（同性食神，异性伤官）
我克者为财（同性偏财，异性正财）
克我者为官杀（同性七杀，异性正官）
生我者为印（同性偏印，异性正印）
```

### 4.7 大运规则
- 阳年男/阴年女 → 顺排（从月柱往下顺数）
- 阴年男/阳年女 → 逆排（从月柱往上逆数）
- 起运岁数 = 距最近节气天数 × 3（天）÷ 1 = 岁数
- 每步大运管10年

### 4.8 神煞推算
- 天乙贵人、桃花、驿马、羊刃、空亡等
- 以日干或年干为基准查找
